# Card Component

The `Card` component offers a flexible and customizable way to display content. Below are the details of the properties and usage examples.

## Examples

### Basic Card

import Card from '../../components/Card';
import Button from '../../components/Button';

<Card
  title="Card title"
  text="Some quick example text to build on the card title and make up the bulk of the card's content."
>
  <Button variant="primary" size="medium" className="mt-4">
    Go Somewhere
  </Button>
</Card>

## Props

| Prop      | Type   | Default          | Description                                                                                      |
| --------- | ------ | ---------------- | ------------------------------------------------------------------------------------------------ |
| title     | string | -                | The title of the card.                                                                           |
| subtitle  | string | -                | The subtitle of the card.                                                                        |
| text      | string | -                | The main text content of the card.                                                               |
| imgSrc    | string | -                | The URL of the image to display in the card.                                                     |
| imgAlt    | string | 'Card image cap' | The alt text for the image.                                                                      |
| header    | string | -                | The content of the card's header.                                                                |
| footer    | string | -                | The content of the card's footer.                                                                |
| textAlign | string | 'left'           | The text alignment (`left`, `center`, `right`).                                                  |
| variant   | string | 'basic'          | The style variant of the card (`basic`, `primary`, `secondary`, `success`, `danger`, `warning`). |
| outline   | bool   | false            | Whether to show a border instead of a solid background.                                          |
| className | string | ''               | Additional CSS classes for the card.                                                             |
| children  | node   | -                | Additional content inside the card body.                                                         |
