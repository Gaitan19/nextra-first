## Sizes

Fancy larger or smaller buttons? Use the `size` prop with `lg` or `sm` for additional sizes.

### Examples

<div className="mb-3">
  <Button variant="primary" size="large" className="me-2">
    Large Primary
  </Button>

{' '}

<Button variant="primary" size="small" className="me-2">
  Small Primary
</Button>

</div>

### Code Examples

```js copy
<div className="mb-3">
  <Button variant="primary" size="lg" className="me-2">
    Large Primary
  </Button>
  <Button variant="primary" size="sm" className="me-2">
    Small Primary
  </Button>
</div>
```

## Props

| Prop      | Type   | Default   | Description                                                                                                                                                      |
| --------- | ------ | --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| children  | node   | -         | The content inside the button, typically text or other elements.                                                                                                 |
| variant   | string | 'primary' | Determines the button's color scheme and styling. Available options are `primary`, `secondary`, `success`, `danger`, `warning`, `info`, `light`, `dark`, `link`. |
| outline   | bool   | false     | Whether to show a border with text color instead of a solid background.                                                                                          |
| size      | string | 'medium'  | Determines the button's size. Available options are `small`, `medium`, `large`.                                                                                  |
| className | string | ''        | Additional CSS classes to apply to the button for further customization.                                                                                         |
